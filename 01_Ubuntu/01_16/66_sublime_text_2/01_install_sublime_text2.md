# install
```
sudo add-apt-repository ppa:webupd8team/sublime-text-2
sudo apt-get update
sudo apt-get install sublime-text
```

# uninstall
```
sudo apt-get remove --auto-remove sublime-text
sudo add-apt-repository --remove ppa:webupd8team/sublime-text-2
```
