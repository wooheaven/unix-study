# rsync option -aqxP
```bash
$ rsync --help | egrep "\-a,|\-q,|\-x,|\-P"
 -a, --archive               archive mode; same as -rlptgoD (no -H)
 -q, --quiet                 suppress non-error messages
 -x, --one-file-system       don't cross filesystem boundaries
 -P                          same as --partial --progress
```
